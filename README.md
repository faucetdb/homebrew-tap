## Install

```bash

```

## Upgrade

```bash
brew update
brew upgrade faucet
```

## About

This tap is automatically updated by [GoReleaser](https://goreleaser.com/) when new Faucet versions are released.

For more information, visit [faucet.dev](https://faucet.dev).
